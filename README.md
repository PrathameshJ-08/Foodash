# Food Dash

> Introducing Food Dash, an innovative food ordering application made with React.js. Elevate your culinary experience as Food Dash seamlessly connects users with their preferred dishes. What distinguishes Food Dash is its unparalleled integration with the live Swiggy API. With Redux handling state, React Router DOM ensuring easy navigation, and Tailwind CSS crafting a visually appealing interface alongside Lazy Loading. Also, it utilizes Parcel for efficient bundling and Firebase for user authentication. Explore the features below.

---

## ⚙ Tech Stack

- **React:** Empowering dynamic and interactive user interfaces with unparalleled flexibility.
- **Parcel:** Streamlining development with efficient bundling and package management.
- **Tailwind CSS:** Elevating design aesthetics through a visually captivating and responsive framework.
- **Redux Toolkit:** Orchestrating application state for a scalable and well-organized architecture.
- **React Router:** Facilitating seamless navigation, ensuring a smooth user experience.
- **Firebase:** Providing robust user authentication for enhanced security and personalized interactions.

---

## 🎯 Key Features

1. 🌏 **_Real-time Swiggy API Integration:_** Food Dash provides users with up-to-the-minute information on menus, availability, and promotions, ensuring accurate and reliable details for informed dining choices.

2. 🌐 **_Location-Aware Restaurant Suggestions:_** Uses geolocation to suggest nearby restaurants based on the user's current location. This ensures that users discover new and exciting dining options close to them.

3. 🔐 **_Firebase Authentication:_** Ensure secure and personalized interactions with Firebase authentication, adding a layer of robust user authentication to Food Dash.

4. 💤 **_Lazy Loading & Chunking:_** Implementing lazy loading and chunking for optimization, Food Dash significantly reduces app loading time, delivering a swift and responsive user experience.

5. 🔩 **_Custom Hook:_** Utilizing custom hooks, Food Dash enhances code reusability and maintainability, fostering a more efficient development process and a streamlined architecture.

6. ♻ **_State Management with Redux:_** Ensuring a scalable and organized application state with Redux,

7. 🔍 **_Search Functionality:_** Easily find your desired restaurants or dishes with Food Dash's intuitive search feature.

8. 🎯 **_Filtering Options:_** Tailor your browsing experience with filtering options, allowing you to discover restaurants based on specific categories or preferences.

9. 📌 **_Seamless Navigation:_** With React Router DOM, the app provides a smooth and intuitive navigation experience, enabling users to effortlessly explore different sections, from discovering restaurants to tracking orders, without disruptions.

10. 📜 **_Detailed Restaurant Information:_** Explore comprehensive menus with real-time updates for each restaurant. Get detailed information about dishes, prices, and special offers.

11. 🛒 **_Seamless Cart Management:_** Effortlessly manage your cart with features like item quantity adjustments, easy removal, and a clear overview of the total bill.

12. ✨ **_Shimmer UI:_** Enhancing user interface transitions with shimmer UI, Food Dash provides a visually appealing loading effect during data retrieval, creating a polished and engaging experience for users while waiting for content to load.

13. 🚧 **_Error 404 Page:_** In the event of a page not found or API failure, Food Dash gracefully presents a user-friendly Error 404 page, ensuring a smooth and informative experience for users.

14. 📱 **_Completely Responsive:_** Enjoy a seamless experience across various devices as Food Dash ensures complete responsiveness, adapting to different screen sizes and resolutions.

---

## Getting Started

To run Foodash locally, follow these steps:

1. Clone the repository: `git clone [https://github.com/PrathameshJ-08/Foodash]`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

Visit [Foodash](https://foodash.vercel.app/) to see the application in action.

Feel free to explore the codebase, contribute, and enhance the Foodash experience!
